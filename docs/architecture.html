YP-Video

An end-to-end system that automatically detects rally segments in volleyball match videos using a VLM-bootstrapped training pipeline.

💡

Core Idea: VLM-Bootstrapped Data Flywheel

Vision-Language Models (VLMs) can understand video content without task-specific training, but they're slow and expensive to run on every new video. Temporal Action Detection (TAD) models are fast at inference, but require labeled training data that's expensive to create from scratch.

This system bridges the gap: use a VLM to auto-generate rally labels, let humans review and correct them, then train a lightweight TAD model on the curated data. The result is a fast, specialized model — built without manual annotation from zero.

Pipeline

Two Phases, One Feedback Loop

Phase A bootstraps high-quality training data using a VLM + human review. Phase B trains a fast model that can process new videos in seconds instead of minutes. Each round of new predictions can feed back into annotation, improving the model over time.

System

Architecture

A unified web dashboard orchestrates every step. A single A100 GPU is shared between the VLM server and TAD training through an async lock — no manual GPU management needed.

🌐 Web Dashboard

  • FastAPI async server
  • SPA frontend for all pipeline steps
  • Real-time SSE job progress
  • Video annotation editor

⚡ GPU Scheduler

  • Single A100-80GB GPU
  • vLLM serves Qwen3-VL for detection
  • TAD uses GPU for training & inference
  • Async lock prevents contention

☁️ Cloud Sync

  • Cloudflare R2 object storage
  • Auto-backup after each pipeline step
  • Local-first, cloud as durable store
  • Shareable annotation artifacts

👁️ VLM Path (Slow, Accurate)

  • Qwen3-VL 8B via vLLM (OpenAI-compatible API)
  • 6s sliding window with 2s stride
  • Parallel batch inference
  • Voting + moving average post-processing

🎯 TAD Path (Fast, Learned)

  • V-JEPA 2.1 ViT-B feature encoder (BF16 + torch.compile)
  • ActionFormer temporal detection head
  • Feature caching for instant re-inference
  • Auto rally clip export
